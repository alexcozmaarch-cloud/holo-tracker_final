Connect to Google Sheets
1
Create a Google Sheet
Go to sheets.google.com, create a blank spreadsheet, name it HOLO TCG Tracker.
2
Open Apps Script & paste code
In your sheet: Extensions → Apps Script. Delete everything, paste the code below, then Save (Ctrl+S).
